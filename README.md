# Mobiadd - Quick Installation

**One-line installation command for Linux and macOS:**
## Quick Install
```bash
curl -fsSL https://raw.githubusercontent.com/andalandaloo/setup/refs/heads/main/install.sh | sudo bash
```
## MacOS
```bash
curl -fsSL https://raw.githubusercontent.com/andalandaloo/setup/refs/heads/main/setup_macos.sh | sudo bash
```

## Linux
```bash
curl -fsSL https://raw.githubusercontent.com/andalandaloo/setup/refs/heads/main/setup.sh | sudo bash
```


This will automatically install Mobiadd with all dependencies.

**After installation:**
- Access: `http://YOUR_SERVER_IP:4000`
- Username: `admin`
- Password: `****`
